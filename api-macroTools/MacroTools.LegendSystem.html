Capital

CapitalManager

Legend

A Legend is a wrapper for unique heroes. A Legend can continue to exist even if the unit it describes does not. A Legend might have other units it relies on to survive. If so, when it dies, it gets removed if those units are not under control. There is a dummy ability to represent this.
